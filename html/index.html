Projetos :

O Resume API é uma API projetada para ajudar estudantes a organizar e revisar seus resumos de maneira eficiente. A plataforma permite que os alunos criem, editem e organizem resumos de diferentes matérias, tudo de forma prática e segura. Com funcionalidades como cadastro de usuários, login seguro e filtragem de resumos por matéria, o Resume API facilita o acesso à informação e otimiza o processo de estudo.

O Bookstore API é uma API backend desenvolvida para permitir que livrarias gerenciem seu estoque de livros de maneira eficiente e segura. A plataforma oferece funcionalidades completas para o gerenciamento de livros e usuários. Com funcionalidades como o registro de contas, login seguro com autenticação e criptografia de senhas, a API permite que os usuários criem, editem, excluam e busquem livros de forma simples e prática. O Bookstore API facilita o processo de gestão de livros e garante a segurança dos dados dos usuários.
